# SafariCharge Solar Dashboard

A Next.js dashboard for monitoring, managing, and analyzing solar energy systems.

## Key features

- Solar performance visualizations (charts/tables)
- Data upload/download workflow
- Report generation endpoints
- Prisma-backed database layer

## Tech stack

- Next.js (App Router) + TypeScript
- Tailwind CSS + shadcn/ui
- Prisma

## Run locally

`ash
npm install
npm run dev
`

## Environment

Copy .env.example to .env and set:

- DATABASE_URL (required)
- GEMINI_API_KEY (optional)
